# Changelog

모든 주요 변경 사항이 이 파일에 기록됩니다.

## [0.1.0] - 2025-03-31

### 추가
- 초기 MVP 구현 완료
  - 텍스트 메시지 작성 및 저장 기능
  - 날짜/시간 기반 열람 제한 시스템
  - 암호화된 URL 공유 메커니즘
  - 카운트다운 타이머 UI
- CryptoJS 라이브러리 통합 (클라이언트 측 AES 암호화)
- LocalStorage를 활용한 메타데이터 참조 저장
- 심플하고 모던한 UI 디자인 (반응형)
- 봉투 생성 및 열람 탭 구현
- 링크 복사 기능
- 기본 오류 처리 및 유효성 검증
